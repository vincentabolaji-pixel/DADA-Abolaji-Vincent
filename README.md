Project Structure

The project contains the following files:


index.html

The `index.html` file contains the structure of the homepage. It includes:

- Header: Displays the name and tagline.
- About Me Section: A brief description about the user.
- Skills Section: A list of frontend skills (HTML, CSS, JavaScript, etc.).
- Contact Section: Contains links to dummy contact information (LinkedIn, GitHub, Email).

style.css

The style.css file defines the styles for the webpage. It includes:

- Background Color: A pleasant background color for the page.
- Hover Effect: A hover effect on links that changes their color when hovered over.
- Section Styling: Padding and margin adjustments for better layout spacing.
- Header Styling: Centered text for the header section.

Features

- Clean and simple design
- Responsive layout for mobile devices
- Hover effect on links
- Easy to modify for personal use
